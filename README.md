# Campings Europe — PWA

## Déploiement sur GitHub Pages (gratuit, 5 minutes)

### Étape 1 — Créer un compte GitHub
Allez sur https://github.com et créez un compte gratuit.

### Étape 2 — Créer un dépôt
1. Cliquez sur "New repository"
2. Nom : `campings-europe`
3. Cochez "Public"
4. Cliquez "Create repository"

### Étape 3 — Uploader les fichiers
1. Dans votre dépôt, cliquez "uploading an existing file"
2. Glissez-déposez les 3 fichiers : index.html, manifest.json, sw.js
3. Cliquez "Commit changes"

### Étape 4 — Activer GitHub Pages
1. Allez dans Settings → Pages
2. Source : "Deploy from a branch"
3. Branch : main / (root)
4. Cliquez Save
5. Attendez 1 minute, votre URL sera :
   https://VOTRE-PSEUDO.github.io/campings-europe/

### Étape 5 — Installer sur Android
1. Ouvrez Chrome sur votre téléphone
2. Allez sur votre URL GitHub Pages
3. Chrome affiche une bannière "Ajouter à l'écran d'accueil"
   (ou Menu ⋮ → "Ajouter à l'écran d'accueil")
4. Confirmez — l'icône ⛺ apparaît sur votre bureau Android !

## Utilisation
- Lancez l'app : elle télécharge tous les campings d'Europe (~10 min première fois)
- Les données sont mises en cache 48h dans le navigateur
- Cliquez sur un marqueur pour voir nom, étoiles, site web, téléphone
- Bouton Légende pour voir le code couleur
- Bouton Actualiser pour forcer un nouveau téléchargement
